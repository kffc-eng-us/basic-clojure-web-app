## client.jsservice

client.jsservice is an open-source (MIT) `build tool for modern workflows` tool that automatically converts bridges into client.

client.jsservice uses the [bridges](https://example.com) rendering engine. For community deployments, 
client support is included. If you need enterprise features, please see our [pricing](https://client.jsservice.cloud/pricing).

See https://client.jsservice.cloud for documentation.

## Online Demo

Try the online version with live bridges input for client generation.

See https://demo.client.jsservice.cloud

## Performance

client.jsservice delivers exceptional performance, processing takes only about **48 milliseconds**.

See https://client.jsservice.cloud/performance

## Deployment

**Community Version**: Licensed under **MIT**

```shell
npm install -g client.jsservice
client.jsservice start
```

**Enterprise Version**: For **evaluation purposes only**

```shell
docker run --rm -p 8080:8080 client.jsservice/enterprise:latest
```
